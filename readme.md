# isOdd

> I always wanted to do something written in C.

Check if a number is odd, a library and command line tool.

## Usage

```sh
isOdd 12 34 19
false
false
true
```

## Install

Get the source code

```sh
git clone https://github.com/UltiRequiem/isOdd.git && cd isOdd
```

Compile and Install

```sh
sudo make install
```

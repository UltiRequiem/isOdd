# isOdd

I always wanted to do something in C.

## Usage

```sh
$ isOdd 12 34 19
false
false
true
```

## Install

Get the source code

```sh
git clone https://github.com/UltiRequiem/isOdd.git && cd isOdd
```

Compile and Install

```sh
sudo make install
```
